
Steps for VScode Windows 11 for c++ to add icon (.ico)to .exe

1. Download or create .ico
2. In notepad create a file called resources.rc in same directory
3. in .rc file enter "IDI_MYICON ICON "images\icons\flame_light.ico" // appropriate file path to icons
4. Run windows command in same directory of .rc and .ico file - rc.exe /fo resource.res myresource.rc
5. Modify the tasks.json file to add the linker command
"args": [
                "magian.cpp",
                "resource.res", //include this line
                "-o",
                "magian.exe"
            ],
